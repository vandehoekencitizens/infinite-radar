# Infinite Radar ✈️

A real-time Infinite Flight tracking web app with 3D globe and glass cockpit.

## Features
- 3D globe (Cesium)
- Live aircraft tracking
- Smooth interpolation (no jumping)
- Flight trails
- Server selection (Expert / Training / Casual)
- Radar mode & IFE mode
- Simulated glass cockpit (PFD + ND)

## Setup

1. Get an API key from Infinite Flight
2. Replace API_KEY in app.js

## Run

Open index.html in browser OR deploy to GitHub Pages.

## Controls

- Select server (top right)
- Click aircraft to track
- Switch to IFE mode to follow aircraft

## Notes

Glass cockpit is simulated using available API data.
